# Terminal UI Specification

## Framework: Textual (textualize.io)

## Screens

1. **Dashboard** -- Live scan results, target map, phase progress bar, streaming
   LLM reasoning, attack chain visualization, immutable audit log
2. **Findings Detail** -- Deep-dive into a single CVE with exploit options
3. **HITL Approval** -- Modal dialog for exploit authorization
4. **Report Viewer** -- Generated pentest report display and export
5. **Configuration** -- Target scope, rules of engagement, tool settings
6. **Test Lab** -- Safe local vulnerable targets for safe testing
